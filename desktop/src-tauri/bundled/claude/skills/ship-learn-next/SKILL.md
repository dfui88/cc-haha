# Ship / Learn / Next

## 核心理念
快速交付 → 学习反馈 → 迭代改进，循环往复。

## 工作流

### Ship (交付)
- 最小可行功能，快速上线
- 不要追求完美
- 80% 即可发布

### Learn (学习)
- 收集真实使用反馈
- 分析数据指标
- 总结经验教训

### Next (下一步)
- 根据学习结果规划
- 确定下一轮优先级
- 继续 Ship

## 原则
- 速度优于完美
- 真实反馈优于假设
- 小步快跑
