# Superpowers - Vibecoding 必备

## 核心理念
编码前用 Superpowers 模式充分思考，让编码过程更流畅高效。

## 工作流

1. **Idea → Requirements**
   - 把模糊想法转化为明确需求
   - 定义成功标准

2. **Requirements → Tasks**
   - 拆解为具体任务
   - 估算工作量
   - 确定依赖关系

3. **Tasks → Plan**
   - 排列执行顺序
   - 识别风险点
   - 准备技术方案

4. **Plan → Code**
   - 开始 vibecoding
   - 按任务逐项实现
   - 边做边验证

## 适用场景
- 新项目启动
- 新功能开发
- 重构规划
- Bug 修复前的根因分析
