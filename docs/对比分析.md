## 九、已发现并修复的问题

| 问题 | 文件 | 严重程度 | 状态 |
|------|------|---------|:----:|
| `@tauri-apps/api` 在 optionalDependencies → WebView 运行时找不到模块 | `package.json` | **CRITICAL** | 已修复 |
| `/* @vite-ignore */` 阻止 Vite 打包 Tauri API 模块 | 5 个文件中的动态 import | **CRITICAL** | 已修复 |

### 修复内容

**`package.json`** — 将 `@tauri-apps/api`、`@tauri-apps/plugin-dialog`、`@tauri-apps/plugin-process`、`@tauri-apps/plugin-shell`、`@tauri-apps/plugin-updater` 从 `optionalDependencies` 移到 `dependencies`：

```json
"dependencies": {
  "@tauri-apps/api": "2.11.0",
  "@tauri-apps/plugin-dialog": "^2.7.1",
  "@tauri-apps/plugin-process": "^2.3.1",
  "@tauri-apps/plugin-shell": "^2.3.5",
  "@tauri-apps/plugin-updater": "^2.10.1",
}
```

**涉及文件（共 5 处移除 `/* @vite-ignore */`）：**
- `src/lib/desktopRuntime.ts` — 导入 `@tauri-apps/api/core`
- `src/api/terminal.ts` — 导入 `@tauri-apps/api/core` 和 `@tauri-apps/api/event`
- `src/components/layout/AppShell.tsx` — 导入 `@tauri-apps/api/event`
- `src/components/layout/Sidebar.tsx` — 导入 `@tauri-apps/api/window`
- `src/components/layout/TabBar.tsx` — 导入 `@tauri-apps/api/window`
